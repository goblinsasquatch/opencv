Live Streaming face detection

An OpenCV Haar Cascade face and eye detection model, running on a Raspberry Pi using a NOIR camera, and served in a Flask Application.

  • No infrared filter is fitted to the camera sensor, resulting in a purple tint in daylight. This allows low-light operation.
  • FPS is calculated and displayed.
  • Faces and eyes highlighted.
  • Each frame is read, encoded into a bitstream, and yielded into an <img> tag.
  • This html is then viewed on a remote client.
